What actually got built

01 / PACKAGE

A movable Pi package

Install locally, from npm, or from Git. Pi discovers the bundled skill and prompts through package.json.

02 / ENGINE

A strict product loop

It refuses vague outcomes, prevents overbuilding, delegates bounded tasks, and advances only when phase evidence passes.

03 / OBSERVATION

A live control room

The dashboard reads real state, Git, worktrees, agent status, and logs. It never needs your API key.

Important: “zero intervention” begins only after the brief is executable. Before kickoff, the system must resolve missing KPI, acceptance criteria, constraints, and authority. After kickoff, it keeps going until shipped or genuinely blocked.
